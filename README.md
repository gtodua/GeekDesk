# GeekDesk
小巧、美观的桌面快速启动工具

## 开发框架
- wpf
- .net 4.7.2
- [HandyControl](https://github.com/HandyOrg/HandyControl)

## 全局热键 鼠标跟随 快速启动 随时随地
- 支持自定义热键
- 支持鼠标跟随
